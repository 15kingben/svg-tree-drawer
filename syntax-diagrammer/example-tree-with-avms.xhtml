Example Phrase Structure Tree with AVM Nodes

This example demonstrates the functionality of the SVG Tree Drawer. What follows is the phrase structure tree of “They appeared close to me.” Currently, you must use Firefox to view this tree properly, because only this browser currently supports the combination of XHTML, XSLT, XML, MathML, and SVG (with foreignObject). Otherwise, see rendering of this page printed to PDF.

Click on a node to collapse/expand it into a triangle. Shift+Click on a node whose children don't have even heights and they will extend down to be the same level.
